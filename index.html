LEXI Documentation

LEXI is a Python package for the analysis of data from the LEXI imager aboard the Blue Ghost Mission by FireFly Aerospace. The package provides functions for the analysis of cdf files that we receive from the LEXI imager.

The package is developed by the LEXI team at Boston University and is open-source. The source code is available on GitHub under the GNU General Public License v3.0.

Note

The package is still under development and might be a bit buggy to use. Please report any bug that you might find to us. The documentation is also a work in progress and is being continuously updated.
